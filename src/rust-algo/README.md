## Rust 前置知识提醒

文档中的代码会用到以下 Rust 语法，如果你还不熟悉，建议先快速了解：

| 语法 | 例子 | 含义 |
|------|------|------|
| 闭包 | `\|x\| x + 1` | 匿名函数，`\|参数\|` 后跟函数体 |
| 模式解构 | `for &x in arr` | `&x` 把 `&i32` 解包成 `i32` |
| `if let` | `if let Some(v) = opt` | 模式匹配的简写 |
| `Option` | `Some(值)` / `None` | Rust 的"有或没有"，替代 null |
| `.unwrap()` | `opt.unwrap()` | 强制取出 Some 中的值，None 时会 panic |
| `as` 转换 | `x as usize` | 类型转换（注意：负数转 usize 会出错！）|

> 详见 [01_复杂度分析与基础.md](./01_复杂度分析与基础.md) 的"Rust 基础语法速查"部分。

## 学习建议

1. **每天至少刷 1-2 题**：保持手感，量变产生质变
2. **先看文档再刷题**：理解算法本质后再去实践，效率翻倍
3. **总结模板**：每个专题学完后，自己手写一遍模板代码
4. **回顾错题**：做错的题标记下来，隔一周再做一次
5. **循序渐进**：不要跳着学，前面的是后面的基础
