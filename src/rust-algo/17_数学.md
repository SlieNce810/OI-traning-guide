# 17 - 数学

> 数论、组合数学、快速幂——竞赛中的数学武器。

## 一、基础知识

### 1.1 核心定义

算法竞赛中的数学主要涉及以下几个领域：

- **快速幂**：用 O(log n) 时间计算 `a^n mod m`，利用幂次的二进制拆分
- **数论**：最大公约数（欧几里得）、素数筛、模逆元、费马小定理
- **组合数学**：排列组合计数、二项式系数、容斥原理
- **概率与期望**：独立事件的期望线性可加

### 1.2 工作原理

- **快速幂**：`a^n = (a^(n/2))^2`，n 为偶；`a^n = a × (a^((n-1)/2))^2`，n 为奇。每次将指数减半。
- **GCD**（辗转相除法）：`gcd(a,b) = gcd(b, a mod b)`，直到 b=0 返回 a。
- **筛法**：埃氏筛 O(n log log n)，欧拉筛 O(n)。标记合数的倍数。
- **模运算**：所有运算对 mod 取模，除法转为乘模逆元（费马小定理：`a^(p-2) ≡ a^(-1) mod p`，p 为质数）。

### 1.3 适用场景与前提

| 场景 | 技术 | 复杂度 |
|------|------|--------|
| 大数幂取模 | 快速幂 | O(log n) |
| 最大公约数 | 辗转相除 | O(log n) |
| 质数判断 | 筛法 或 Miller-Rabin | O(√n) 到 O(log³ n) |
| 组合数 C(n,k) | 预处理阶乘 + 逆元 | O(n) 预处理，O(1) 查询 |
| 容斥计数 | 容斥原理 | O(2^k)，k 为条件数 |

**前提**：加法、减法、乘法对任何模数（质数或合数）都有效。但**模逆元**（即"除法"）需要 `gcd(a, mod) = 1`——费马小定理求逆元要求 mod 是质数，但扩展欧几里得算法适用于任何与 a 互质的模数。LeetCode 最常见的 `10^9+7` 是质数，费马小定理就够用了。大数操作必须用 i64/i128，防止溢出。筛法空间 O(n)，n 太大时不可行。

### 1.4 优缺分析

**优点：**
- 数论算法时间复杂度极低（O(log n) 或 O(√n)）
- 模运算保证结果不溢出，答案可控
- 数学题通常代码量少，重在推导

**缺点：**
- 推导过程是主要难点——知道公式才能写代码
- 模逆元依赖质数模，部分题目模数为合数时需扩展欧几里得
- 组合数预处理需要 O(n) 空间，n 较大时限制严格
- 溢出风险高——每一步乘法都可能溢出 i64，需合理使用 i128 或 mod 取模

## 二、快速幂

### 算法本质

**大白话：计算 a^n，不用乘 n 次，只需要乘 log(n) 次。**

原理：利用 `a^n = (a^(n/2))^2`，把幂次不断减半。

```rust
/// 快速幂：计算 base^exp % modp
fn power(mut base: i64, mut exp: i64, modp: i64) -> i64 {
    let mut result = 1i64;
    base %= modp;
    while exp > 0 {
        if exp & 1 == 1 {
            result = result * base % modp;
        }
        base = base * base % modp;
        exp >>= 1;
    }
    result
}
```

**过程模拟（计算 2^10）：**
```
2^10 = (2^5)^2
2^5  = 2 × (2^2)^2
2^2  = (2^1)^2
只需要 4 次乘法，而不是 10 次！
```

## 三、GCD 和 LCM

```rust
fn gcd(a: i64, b: i64) -> i64 {
    if b == 0 { a.abs() } else { gcd(b, a % b) }
    // 注意：用 abs() 保证结果为正数
}

fn lcm(a: i64, b: i64) -> i64 {
    a / gcd(a, b) * b // 先除后乘，防溢出
}
```

## 四、质数

### 判断质数

```rust
fn is_prime(n: i64) -> bool {
    if n < 2 { return false; }
    let mut i = 2;
    while i * i <= n {
        if n % i == 0 { return false; }
        i += 1;
    }
    true
}
```

### 埃氏筛（Sieve of Eratosthenes）

```rust
/// 筛出 [0, n] 中所有质数
fn sieve(n: usize) -> Vec<bool> {
    let mut is_prime = vec![true; n + 1];
    is_prime[0] = false;
    if n >= 1 { is_prime[1] = false; }

    let mut i = 2;
    while i * i <= n {
        if is_prime[i] {
            let mut j = i * i;
            while j <= n {
                is_prime[j] = false;
                j += i;
            }
        }
        i += 1;
    }
    is_prime
}

// 获取质数列表
fn primes_up_to(n: usize) -> Vec<usize> {
    let is_prime = sieve(n);
    (2..=n).filter(|&i| is_prime[i]).collect()
}
```

### 质因数分解

```rust
fn factorize(mut n: i64) -> Vec<(i64, i32)> {
    let mut factors = vec![];
    let mut d = 2;
    while d * d <= n {
        if n % d == 0 {
            let mut count = 0;
            while n % d == 0 {
                n /= d;
                count += 1;
            }
            factors.push((d, count));
        }
        d += 1;
    }
    if n > 1 {
        factors.push((n, 1));
    }
    factors
}
```

## 五、模运算

在竞赛中，答案经常要求对 10^9+7 取模。

```rust
const MOD: i64 = 1_000_000_007;

// 模运算基本规则：
// (a + b) % m = ((a % m) + (b % m)) % m
// (a * b) % m = ((a % m) * (b % m)) % m
// (a - b) % m = ((a % m) - (b % m) + m) % m  // 注意加 m 防负数

/// 模逆元：a^(-1) mod p = a^(p-2) mod p （费马小定理，p 为质数）
/// 原理：费马小定理说 a^(p-1) ≡ 1 (mod p)，
/// 因此 a * a^(p-2) ≡ 1 (mod p)，即 a^(p-2) 就是 a 的逆元
fn mod_inverse(a: i64, p: i64) -> i64 {
    power(a, p - 2, p)
}
```

> 🔑 **费马小定理（Fermat's Little Theorem）**：若 p 是质数且 a 不被 p 整除，则 `a^(p-1) ≡ 1 (mod p)`。
> 
> 通俗理解：在模 p 的意义下，a 的 `p-1` 次方等于 1。这使我们可以通过 `a^(p-2) mod p` 来求 a 的逆元，因为 `a × a^(p-2) = a^(p-1) ≡ 1 (mod p)`。

```rust
>
> ⚠️ **重要限制**：这要求 p 是质数。如果模数是合数（如 10^9），需要用**扩展欧几里得算法**求逆元。LeetCode 中绝大多数题目用的 `10^9+7` 是质数，所以费马小定理够用。
>
> ⚠️ **模运算常见错误**：
> ```rust
> // 错误：没有对中间结果取模，可能溢出 i64
> let bad = (a * b) % MOD;  // a*b 可能溢出！
> 
> // 正确：先取模再乘
> let good = (a % MOD) * (b % MOD) % MOD;
> // 或使用 i128 中间类型：
> let good = ((a as i128) * (b as i128) % MOD as i128) as i64;
> ```

/// 模除法：(a / b) % p = a * b^(-1) % p
fn mod_div(a: i64, b: i64, p: i64) -> i64 {
    (a % p) * (mod_inverse(b, p) % p) % p // 注意加括号确保运算顺序
}
```

## 六、组合数学

### 组合数 C(n, k)

```rust
/// 预计算阶乘和逆元，支持快速求组合数
struct Comb {
    fact: Vec<i64>,     // 阶乘
    inv_fact: Vec<i64>, // 阶乘的逆元
    modp: i64,
}

impl Comb {
    fn new(n: usize, modp: i64) -> Self {
        let mut fact = vec![1i64; n + 1];
        for i in 1..=n {
            fact[i] = fact[i - 1] * i as i64 % modp;
        }
        let mut inv_fact = vec![1i64; n + 1];
        inv_fact[n] = power(fact[n], modp - 2, modp);
        for i in (0..n).rev() {
            inv_fact[i] = inv_fact[i + 1] * (i + 1) as i64 % modp;
        }
        Comb { fact, inv_fact, modp }
    }

    /// C(n, k) = n! / (k! * (n-k)!)
    fn comb(&self, n: usize, k: usize) -> i64 {
        if k > n { return 0; }
        let m = self.modp;
        // 注意用括号确保每步都取模，防止溢出
        self.fact[n] % m * (self.inv_fact[k] % m) % m * (self.inv_fact[n - k] % m) % m
    }
}
```

### 常用公式

| 公式 | 含义 |
|------|------|
| C(n,k) = C(n-1,k-1) + C(n-1,k) | 杨辉三角 |
| C(n,0) + C(n,1) + ... + C(n,n) = 2^n | 子集总数 |
| A(n,k) = n! / (n-k)! | 排列数 |
| C(n,k) = C(n, n-k) | 对称性 |

### 容斥原理

**大白话：要数"至少满足一个条件"的方案数，用各条件的方案数相加，再减去同时满足两个的，再加上同时满足三个的……**

|A∪B∪C| = |A| + |B| + |C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C|

```rust
fn my_pow(x: f64, n: i32) -> f64 {
    let mut result = 1.0f64;
    let mut base = x;
    let mut exp = n as i64;
    if exp < 0 {
        base = 1.0 / base;
        exp = -exp;
    }
    while exp > 0 {
        if exp & 1 == 1 {
            result *= base;
        }
        base *= base;
        exp >>= 1;
    }
    result
}
```

### 例题：LeetCode 172. 阶乘后的零

> 返回 `n!` 结果末尾连续零的个数（零由因子 2×5 产生，阶乘中因子 5 更稀缺，数 5 即可）。
>
> **示例：** `n = 3` → `0`；`n = 5` → `1`
> **约束：** `0 <= n <= 10^4`

```rust
fn trailing_zeroes(n: i32) -> i32 {
    // 末尾零 = 因子5的个数（因子2总是够的）
    let mut count = 0;
    let mut n = n;
    while n >= 5 {
        n /= 5;
        count += n;
    }
    count
}
```

## 八、测试题

1. **问：** 为什么取模运算中减法要加一个 MOD？

   **答：** 因为 `(a - b) % MOD` 可能得到负数。加上 MOD 后再取模，保证结果为正。

2. **问：** 为什么 10^9+7 是竞赛中最常用的模数？

   **答：** 因为它是质数（可以用费马小定理求逆元），而且两个小于它的数相乘不会超过 i64 的范围。
