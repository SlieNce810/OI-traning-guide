# 第2章 数学基础

如果说第一章教的是"怎么想"，那么本章教的是"用什么算"。算法竞赛中的数学，不是纯数学证明，而是以高效计算为目的的数学工具应用。本章的目标是在最短篇幅内让读者掌握竞赛中最高频出现的数学知识点，并理解它们在具体问题中的作用方式——不是记住公式，而是知道在什么场景下该调用哪个数学工具。

本章九个小节的编排遵循从离散到连续、从基础到进阶的逻辑顺序。前四节构成了一条"计数与分析"的主线：基本计数方法从排列组合和容斥原理开始，这是整个算法竞赛数学的基石——几乎所有"有多少种方案"的问题最终都归结为计数。递推关系一节则揭示了计数问题的另一面：不是所有计数都适合用公式直接表达，很多时候递推（加上DP）是更自然的建模方式，本节通过多叉树遍历等例题展示了递推与区间DP的结合。数论一节是本章内容最丰富的部分，从最大公约数和扩展欧几里得出发，逐步深入到莫比乌斯反演和数论分块——这些工具的威力在于它们能将"看起来要O(N)枚举"的问题压缩到对数级别。组合游戏一节引入的SG函数是博弈问题的统一建模语言：任何公平组合游戏都可以通过计算SG值来判断先手胜败，关键在于将复杂游戏分解为若干独立子游戏的异或和。

中间三节构成了"代数与变换"的主线。概率与数学期望一节关注的是带有随机性的计数问题——这类问题通常需要建立递推关系来求解期望值，核心技巧在于将状态转移的概率转化为求和项。置换及其应用一节则从群论视角审视排列问题：置换可以分解为若干循环，这种分解揭示了许多组合结构的内在对称性——比如一个置换能否表示为另一个置换的平方，取决于其循环长度的奇偶性。矩阵和线性方程组一节展示了矩阵运算在算法竞赛中的两大核心用途：用矩阵快速幂将一维线性递推加速到O(log n)，以及用高斯消元求解线性方程组。

最后两节属于"进阶工具"范畴。快速傅里叶变换（FFT）一节处理的是一个看似简单但天然O(N²)的问题——多项式乘法，FFT通过将多项式从系数表示转换到点值表示，在O(N log N)时间内完成卷积计算。这一思想在字符串匹配、生成函数、大整数乘法等领域有广泛应用。数值方法简介一节则讨论了一类特殊的最优化问题：当目标函数没有显式解析解时，如何用二分法或三分法逼近最优解——核心在于判断函数的单调性（二分）或单峰性（三分）。
