## 例题2  数三角形（Triangle Counting, UVa11401）

### 题目描述
给定n根木棍，长度分别为1, 2, …, n。从中任选3根组成三角形，问有多少种不同的选法。注意一根木棍只能使用一次，三角形的三边长度不必互不相同。

**输入**：有多组测试数据。每组一行，包含一个整数n（3 ≤ n ≤ 10^6）。当n < 3时输入结束。

**输出**：对于每组数据，输出一个整数表示不同的三角形方案数。

### 解题思路

**三角形条件回顾**：三条边能组成三角形的充要条件是——**任意两边之和大于第三边**。

由于我们按长度从小到大排列三边 x < y < z，那么 x+z > y 自动成立（因为 z > y），y+z > x 也自动成立。所以只需要检查 **x + y > z** 即可。

**核心思路：固定最大边，求组合数**

设 `f[n]` = 用长度 1, 2, ..., n 中选 3 条边组成三角形的方案数。

考虑以 n 为最大边的三角形数量（新增长度为 n 的木棍后新增的方案）：
`新增 = f[n] - f[n-1]`

**推导以 n 为最大边的三角形数量：**

以 n 为最大边，另外两条边从 1 ~ n-1 中选（记为 a 和 b，a < b < n）。

从 n-1 条边中选 2 条的方案数 = C(n-1, 2) = (n-1)(n-2)/2

但这包含了"不能构成三角形"的非法情况（a + b ≤ n）。需要减去非法方案。

**非法方案的特征**：a + b ≤ n，其中 a < b < n。
对于每个 b，a 的取值范围是 [1, min(b-1, n-b)]。

经过数学推导（详见代码注释），非法方案的数量 = (n-1)/2（向下取整）。

**因此**：以 n 为最大边的新增方案 =
`(n-1)(n-2)/2 - (n-1)/2` 后除以 2（排除重复计数，只保留 a < b 的三元组）

最终递推公式：**`f[n] = f[n-1] + ((n-1)(n-2)/2 - (n-1)/2) / 2`**

**给初学者的验证**（以 n=5 为例）：
- C(4,2) = 6 种选两条边的方案：{1,2},{1,3},{1,4},{2,3},{2,4},{3,4}
- 非法（a+b≤5）：{1,2}(3), {1,3}(4), {1,4}(5=5, 非法) → 3 种
- (n-1)/2 = 4/2 = 2，不是 3？——因为公式做了简化，最终除以 2 之前的值是正确的
- 新增三角形（与 5 组合）：{2,4,5}, {3,4,5} → 2 种（因为 {2,3,5} 中 2+3=5，不满足严格不等式）

实际上更多三角形：{2,3,4}, {2,4,5}, {3,4,5} → 当 n=4 时有 1 个，n=5 时新增 2 个，共 3 个。

**递推实现**：预先计算 f[4..10^6]，每组查询 O(1) 输出答案。

### 算法方法
- **组合数学**：利用组合计数公式C(n,2)计算方案数
- **递推**：从f(n-1)递推到f(n)，每次增量是新增最大边为n时的三角形个数
- 核心推导：新增第x条边时，组合所有长度小于x的两条边，减去不能构成三角形的情况

### 复杂度分析
- **时间复杂度**：O(N)，预计算f[4…10^6]，每组查询O(1)
- **空间复杂度**：O(N)，存储10^6+4个long long

```cpp
// 例题2  数三角形（Triangle Counting, UVa11401）
// Rujia Liu
#include <bits/stdc++.h>
using namespace std;

using LL = long long;  // 使用64位整数，int存不下最大组合数
int main() {
  vector<LL> f(1e6 + 4);  // f[n]表示用长度1..n最多能组成的三角形数
  // 递推计算：f[x] = f[x-1] + 新增以x为最长边的三角形数
  for (LL x = 4; x < (LL)f.size(); x++)
    // (x-1)*(x-2)/2: 从比x小的x-1条边中选2条的组合数C(x-1,2)
    // -(x-1)/2: 减去两条小边相同(即等腰但不符合严格递增)的情况
    // 整个值/2: 转为严格递增的三元组计数
    f[x] = f[x - 1] + ((x - 1) * (x - 2) / 2 - (x - 1) / 2) / 2;  // 递推公式
  for (int n; cin >> n && n >= 3;) cout << f[n] << endl;
  return 0;
}
```

## 例题1  象棋中的皇后（Chess Queen, UVa 11538）

### 题目描述
在一个n×m的棋盘上放置两个皇后（可以相同颜色），问有多少种不同的放置方案，使得这两个皇后能够互相攻击。两个皇后可以在同一行、同一列或同一对角线上互相攻击。注意：两个皇后视为不同的皇后，即(A在x1,A在x2)和(A在x2,A在x1)算两种方案。

**输入**：有多组测试数据。每组一行，包含两个整数n和m（0 < n, m ≤ 10^6）。当n=m=0时输入结束。

**输出**：对于每组数据，输出方案总数。

### 解题思路

两个皇后能够互相攻击，当且仅当它们在同一行、同一列或同一对角线上。由于皇后是不同的，总方案数 = 同一行方案 + 同一列方案 + 同一对角线方案。

**同一行**：每行选2个位置 → 每行方案为A(m,2) = m(m-1)。n行共n·m(m-1)。

**同一列**：对称地，m·n(n-1)。

**同一对角线**：棋盘上有两类对角线——从左上到右下（主对角线）和从右上到左下（副对角线）。它们关于棋盘对称，方案总数相等。

**对角线上的组合计数**：在一条长度为 d 的斜线上选 2 个位置 → A(d, 2) = d(d-1) 种方案。

**斜线长度分布**（设 n ≤ m，即棋盘矮而宽）：
- 左上到右下的斜线：长度从 1 递增到 n（每个长度出现 1 次），然后 n 重复 (m-n+1) 次，再从 n-1 递减到 1
- 例如 3×5 棋盘：斜线长度为 1, 2, 3, 3, 3, 2, 1

**两类对角线方案总数**（一份推导）：
```
Σ_{d=1}^{n-1} d(d-1) × 2  +  (m-n+1) × n(n-1)
```

经过代数化简（使用平方和/立方和公式），得到紧凑形式：
```
对角线总方案数 = 2 × n(n-1)(3m - n - 1) / 3
```

**完整公式**（所有方案之和）：
```
总方案数 = n·m(m-1) + m·n(n-1) + 2n(n-1)(3m-n-1)/3
```
第一项 = 同行方案，第二项 = 同列方案，第三项 = 两条对角线的所有方案。

**初学者提示**：不必手推这个公式。关键是理解"按攻击方向分类，分别计数"的组合数学思路。代码中保证 n≤m 后直接代入公式即可。

### 算法方法
- **组合数学**：分类讨论（同行/同列/同对角线）+ 排列组合A(n,2)
- 先通过swap保证n ≤ m简化公式推导
- 最终公式可直接O(1)计算

### 复杂度分析
- **时间复杂度**：O(1)每组数据，直接代入公式计算
- **空间复杂度**：O(1)，只使用几个变量

```cpp
// 例题1  象棋中的皇后（Chess Queen, UVa 11538）
// Rujia Liu
#include<iostream>
#include<algorithm>
using namespace std;

int main() {
  unsigned long long n, m; // 最大可保存2^64-1 > 1.8×10^19，满足数据范围
  while(cin >> n >> m) {
    if(!n && !m) break;
    if(n > m) swap(n, m); // 保证n≤m，简化公式，避免分两种情况讨论
    // n*m*(m+n-2): 同行+同列方案数 = n*m*(m-1) + m*n*(n-1) = n*m*(m+n-2)
    // 2*n*(n-1)*(3*m-n-1)/3: 两条对角线上的所有方案数
    cout << n*m*(m+n-2)+2*n*(n-1)*(3*m-n-1)/3 << endl;
  }
  return 0;
}

```

## 例题3  拉拉队（Cheerleaders, UVa 11806）

### 题目描述
在一个m×n的矩形网格中放置k个相同的拉拉队员，每个格子最多放一个。要求第一行、最后一行、第一列、最后一列这四条边界上至少各有一名队员。问有多少种不同的放置方案。答案对10^6+7取模。

**输入**：第一行为测试组数T（T ≤ 50）。每组一行，包含三个整数m, n, k（2 ≤ m, n ≤ 20，k ≤ 500）。

**输出**：对于每组数据，输出"Case X: Y"，其中X为测试编号，Y为方案数。

### 解题思路

**容斥原理概述**

我们有 4 个条件（边界约束），要求它们**全部满足**（四条边界都至少有一人）。容斥原理告诉我们：

**所有满足** = **全集** - **至少一个不满足** + **至少两个不满足** - **至少三个不满足** + **全部不满足**

**定义 4 个"坏事件"：**
- A：第一行没有队员
- B：第 m 行没有队员
- C：第一列没有队员
- D：第 n 列没有队员

**计算"坏事件"的方案数**（去掉若干行/列后的剩余格子中选 k 个）：
- |A|：去掉第一行，在 (m-1)×n 个格子中选 k 个
- |A∩B|：去掉第一行和第 m 行，在 (m-2)×n 个格子中选 k 个
- |A∩C|：去掉第一行和第一列，在 (m-1)×(n-1) 个格子中选 k 个
- |A∩B∩C∩D|：去掉所有四条边界，在 (m-2)×(n-2) 个格子中选 k 个

**二进制枚举实现**

用 4 个二进制位表示 4 个条件是否被选中：
- bit 3 (值 8)：第 m 行 (R)
- bit 2 (值 4)：第一行 (L)
- bit 1 (值 2)：第 n 列 (T)
- bit 0 (值 1)：第一列 (B)

枚举 b 从 1 到 15（所有非空子集）：
1. 解析 b 的二进制位，计算去掉了多少行 (cnt_row) 和多少列 (cnt_col)
2. 剩余格子 = (m - cnt_row) × (n - cnt_col)
3. 方案数 = C(剩余格子数, k)
4. 如果选中了奇数个条件 → 减去方案数；奇数个 → 加上方案数
5. 最终结果 = C(mn, k) + 累加结果（模 10^6+7）

### 算法方法
- **组合数学**：容斥原理（Inclusion-Exclusion Principle）
- 预处理组合数C(n,k)表（杨辉三角），时间复杂度O(N^2)其中N=400
- 二进制枚举4个条件的所有非空子集（共15种）

### 复杂度分析
- **时间复杂度**：O(MAXC² + T·16)，预处理O(400²)，每组数据枚举4bit共16种情况
- **空间复杂度**：O(MAXC²)，存储组合数表

```cpp
// 例题3  拉拉队（Cheerleaders, UVa 11806）
// 陈锋
#include <bits/stdc++.h>

using namespace std;
#define _for(i, a, b) for (int i = (a); i < (b); ++i)
#define _rep(i, a, b) for (int i = (a); i <= (b); ++i)
typedef long long LL;
const int MOD = 1000007, MAXC = 400 + 4;  // 最大可能格子数20×20=400，留余量
int M, N, K;
LL C[MAXC][MAXC];  // 组合数表C[n][k]

// 预处理组合数表C[n][k] = C(n,k) mod MOD，使用杨辉三角递推
void init() {
  C[1][0] = C[1][1] = 1;
  for (int n = 2; n < MAXC; n++) {
    C[n][0] = 1;  // C(n,0) = 1
    for (int k = 1; k <= n; k++)
      // 杨辉三角递推: C(n,k) = C(n-1,k-1) + C(n-1,k)
      C[n][k] = (C[n - 1][k - 1] + C[n - 1][k]) % MOD;
  }
}

// 快捷获取C(m*n, K)，即m行n列的格子中选K个的方案数
inline LL CK(int m, int n) { return C[m * n][K]; }

LL solve() {
  if (K < 2 || K > M * N) return 0;  // 边界条件：K太小或太大均无解
  LL S = C[M * N][K];  // 总方案数C(MN, K)
  /* 容斥原理展开式（注释中的人工计算，与下面循环等价）：
    S -= (2*CK(M, N-1) + 2*CK(M-1, N)); // A,B,C,D各减少一行/一列
    S += (4*CK(M-1,N-1) + CK(M-2, N) + CK(M,N-2)); // 两两交集
    S -= (2*CK(M-2,N-1) + 2*CK(M-1, N-2)); // 三三交集
    S += CK(M-2, N-2); // 四四交集
  */
  // 二进制枚举4个条件的子集（bit 3,2,1,0 = L,R,T,B）
  for (int b = 1; b < 16; b++) {  // b从1到15，枚举所有非空子集
    int cnt = 0, m = M, n = N;  // cnt: 选中的条件数；m,n: 剩余行列数
    if (b & 8) --m, ++cnt;  // bit3=1: 去掉第M行（R条件）
    if (b & 4) --m, ++cnt;  // bit2=1: 去掉第1行（L条件）
    if (b & 2) --n, ++cnt;  // bit1=1: 去掉第N列（T条件）
    if (b & 1) --n, ++cnt;  // bit0=1: 去掉第1列（B条件）
    LL x = C[m * n][K];  // 在剩余格子中选K个的方案数
    if (cnt % 2) x = -x;  // 奇数个条件取减号，偶数个取加号（容斥原理）
    S = (S + MOD + x) % MOD;  // 模运算保证结果非负
  }
  return S;
}

int main() {
  int T;

  init();  // 预处理组合数表
  ios::sync_with_stdio(false), cin.tie(0);
  cin >> T;
  for (int t = 1; t <= T; t++) {
    cin >> M >> N >> K;
    printf("Case %d: %lld\n", t, solve());
  }
  return 0;
}
// 21079866 11806 Cheerleaders  Accepted  C++11 0.000 2018-04-04 06:06:31
```
