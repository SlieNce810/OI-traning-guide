# 算法竞赛入门经典-训练指南-升级版 配套网站

本书是《算法竞赛入门经典-训练指南》的升级版，由**刘汝佳（Rujia Liu）**和**陈锋**编写。

本网站提供书中所有例题和练习题目的完整源代码，按章节组织，涵盖以下六大专题：

| 章节 | 内容 | 题数 |
|------|------|------|
| 第1章 | 算法设计基础 | 32 |
| 第2章 | 数学基础 | 37 |
| 第3章 | 实用数据结构 | 59 |
| 第4章 | 几何问题 | 20 |
| 第5章 | 图论算法与模型 | 34 |
| 第6章 | 更多算法专题 | 23 |

共 **205** 道题目的 C++ 参考实现。

## 代码说明

- 每题一个独立的 `.cpp` 文件，文件名包含来源平台和题号
- 文件头部注释标注了例题编号、题目名称和来源
- 代码可在 UVa、POJ、HDU、CodeChef、SPOJ、Codeforces Gym 等在线评测平台提交
